Первая лабораторная работа

Задание: Создать страничку-отчет и выложить её на хостинг GitHub Pages

Последовательность действий по созданию сайта и выкладывание его на хостинг:

1. Был создан новый репозиторий на сайте github pages
2. Была создана новая папка, в которой находятся все необходимые файлы для работы сайта

Команды для выкладывания сайта на хостинг:

1. git init - создает локальный репозиторий в папке
2. git add . - добавление всех новых файлов в отслеживание
3. git commit -m"commit text" - создания комментария к нашим изменениям
4. git remote add origin ссылка_на_репозиторий - связываем наш локальный репозиторий с удаленным
5. git push -u origin master - отправляем наши изменения на удаленный репозиторий
6. git checkout -b gh-pages - создаем новую ветку gh-pages, которая является хостингом
7. git checkout master - возвращаемся на ветку master
8. git push origin master:gh-pages - отправляем все файлы из ветки msater в ветку gh-pages, тем самым заливая файлы на хостинг
